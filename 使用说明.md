# 傲娇大小姐白菜 - Galgame

一个完整的 Galgame 风格视觉小说游戏！

## 快速开始

```bash
cd /home/yuri/game2
python3 -m http.server 8000
```

然后访问：`http://localhost:8000`

## 故事简介

你是一名通过奖学金考试进入贵族学园——圣光学园的平民学生。

在这里，你遇到了傲娇大小姐**白菜**。

她表面高傲，经常说"哼！"，但内心其实非常温柔...

你的选择将决定你们的关系！

## 游戏特色

### Galgame 标准叙事

- **Prologue + Chapter 结构**
- **旁白 + 内心独白**（使用括号表示动作和心理）
- **分镜式叙事**（"——一周后"等时间跳转）
- **真结局 / 普通结局 / Bad End** 多结局系统
- **选项前缀**（【选项说明】选项内容）

### 傲娇角色塑造

白菜的经典傲娇台词：

- "哼！你以为本小姐会需要你的帮助吗？！"
- "我、我可没说不要...只是暂时借用一下而已！"
- "哼！才、才不是特意为你准备的！"
- "笨蛋！不许看！"
- "你、你这家伙...还真是个笨蛋呢..."（温柔版）

### 3个不同结局

1. **True End "傲娇大小姐的真心"**
   - 帮助白菜 + 接受她的好意
   - 甜蜜的告白结局

2. **Normal End "珍贵的友情"**
   - 帮助白菜 + 拒绝便当
   - 成为朋友

3. **Bad End "陌路人"**
   - 不帮助白菜
   - 从此形同陌路

## 替换角色立绘

### 准备图片

推荐规格：
- 格式：PNG（透明背景）
- 尺寸：800×1600 像素
- 风格：Galgame 立绘风格

### 替换步骤

1. 将图片放在 `images/baicai.png`
2. 打开 [game.js](game.js)，找到第 37-42 行
3. 修改：

```javascript
const characters = {
    baicai: {
        name: '白菜',
        image: 'images/baicai.png',  // ← 改这里
        color: '#32CD32'
    }
};
```

## 剧本结构说明

### Chapter 标题格式

- `Prologue` - 序章
- `Chapter 1 - 命运的邂逅` - 第一章
- `Final Chapter - 樱花树下` - 最终章
- `True End` / `Normal End` / `Bad End` - 结局标题

### 对话格式

```javascript
{
    character: characters.baicai.image,  // 显示角色立绘
    name: '白菜',                        // 角色名
    text: '哼！',                        // 对话内容
    position: 'center'                   // 位置
}
```

### 旁白和动作

```javascript
{
    character: null,
    name: '',
    text: '——一周后',  // 时间跳转
    position: 'center'
}

{
    character: null,
    name: '我',
    text: '（心想）这大小姐真可爱...',  // 内心独白
    position: 'center'
}
```

### 选项格式

```javascript
choices: [
    {
        text: '【帮助她】把外套递给她',
        nextScene: 3
    },
    {
        text: '【离开】默默走开',
        nextScene: 7
    }
]
```


### 音量控制

- 右下角的 🎵 滑块可以调节音量（0-100%）
- 默认音量：30%
- 右上角的 🔊 按钮可以快速静音
- 按 M 键也可以快速静音/取消静音

## ✍️ 编写自己的剧本

### Galgame 叙事技巧

1. **使用时间跳转**：`'——一周后'`、`'——放学后'`
2. **环境描写**：先描写场景氛围
3. **内心独白**：用括号 `（心想）`
4. **动作描写**：`（脸颊泛红）`、`（转身离去）`
5. **音效文字**：`啪！`、`咚咚咚`

### 傲娇角色写作要点

**嘴硬心软**：
- "谢谢你"
- "哼！才、才不是为了感谢你！"

**否定三连**：
- "不、不是...！"
- "才、才没有...！"
- "我、我才不...！"

**经典结尾**：
- "哼！"
- "笨蛋..."
- "...算了"

### 示例场景

```javascript
{
    title: 'Chapter X - 新场景',
    background: 'images/rooftop.jpg',
    scenes: [
        {
            character: null,
            name: '',
            text: '——午休时间，屋顶',
            position: 'center'
        },
        {
            character: characters.baicai.image,
            name: '白菜',
            text: '喂！你怎么会在这里？',
            position: 'center'
        },
        {
            character: null,
            name: '我',
            text: '我...我只是来吹吹风而已...',
            position: 'center'
        },
        {
            character: characters.baicai.image,
            name: '白菜',
            text: '哼！这里可是本小姐的秘密基地！',
            position: 'center'
        },
        {
            character: characters.baicai.image,
            name: '白菜',
            text: '不过...既然你都来了...',
            position: 'center'
        },
        {
            character: characters.baicai.image,
            name: '白菜',
            text: '（小声）就、就特许你待一会儿吧...',
            position: 'center'
        }
    ],
    choices: [
        { text: '【留下】谢谢你，白菜', nextScene: X },
        { text: '【离开】我还是先走吧', nextScene: Y }
    ]
}
```

## 游戏操作

### 鼠标操作
- **点击对话框** / **空格** / **回车** - 继续对话
- **AUTO** - 自动播放
- **SKIP** - 跳过场景
- **SAVE** - 保存进度
- **LOAD** - 加载进度
- **按钮** - 音乐开关
- **滑块** - 音量调节（0-100%）

### 键盘快捷键
- **空格 / 回车** - 继续对话
- **M键** - 快速静音/取消静音

## 创作建议

### 故事结构

1. **Prologue**：设定背景，主角出场
2. **Chapter 1-2**：遇见女主，建立初步印象
3. **Chapter 3-4**：关系升温，好感度提升
4. **Final Chapter**：告白或结局

### 好感度提升路径

1. 帮助她解决困难
2. 接受她的好意
3. 理解她的傲娇
4. 温柔对待她

### 结局设计

- **True End**：好感度 MAX，甜蜜结局
- **Normal End**：好感度中等，友情结局
- **Bad End**：好感度低，错过结局
